Question : Discuss the scope of var, let, and const

VAR:

  • Var is a function scope. Function scope means if a variable is declared using var keyword, it will be accessible throughout the function. var can be redeclared and reassign also.

    • Let and Const:

  • let $ const are block scope. Block scope means if variables declared using let $ const keyword, they are accessible within that particular block. Let can be reassign but can't declare it twice. Const cannot be redeclared and reassign.

    • Question : Tell us the use cases of null and undefined

    null:

  • Null is a type of object. Null is used for missing or unavailable data. if a variable has no value (let sum;) then we can use null. example : (let sum = null);
  • We can reset or clear a variable using null. Example: (let total = 10); (total = null)

    • undefined:

  • we can use undefined to check if a property exists or set optional properties.
  • we can use undefined to represent missing or undefined values in an array.

    • Question : What do you mean by REST API?

    Ans: REST API use standard HTTP methods to perform actions on resources:

  • GET: Retrieve data from the server.
  • POST: Create a new resource on the server.
  • PUT: Update an existing resource or create a new one if it doesn't exist.
  • DELETE: Remove a resource from the server.
  • PATCH: Partially update a resource.
